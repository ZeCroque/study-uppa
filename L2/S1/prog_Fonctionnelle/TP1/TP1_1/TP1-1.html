Programmation Fonctionnelle

 

TP1 : Le logiciel DrRacket (http://racket-lang.org/)

DrRacket est un environnement de programmation basé sur le langage Racket (anciennement PLT Scheme). Nous utilisons pour les travaux pratiques la version 5.3.6 de l'environnement (http://download.racket-lang.org/racket-v5.3.6.html).

1. La fenêtre de DrRacket

La fenêtre de contenu de DrRacket est divisée en deux parties :

  • fenêtre de définitions (en haut): qui permet d'écrire des programmes Scheme ;
  • fenêtre d'interactions (en bas): qui permet d'évaluer des expressions Scheme interactivement.

2. La barre de menus

Fichier

  • Gestion des fichiers : créer un nouveau fichier ; ouvrir un fichier ; sauvegarder le contenu des fenêtres 
  • Fermer : fermer un fichier 
  • Quitter : quitter DrRacket.

Éditer

On y trouve les options d'édition de texte comme : couper/copier/coller, rechercher/remplacer

Montrer

Permet d'avoir soit les deux fenêtres (de définitions et d'interations) soit une seule.

Langage

Permet de choisir le niveau de langage souhaité. Sélectionner le langage d'enseignement R5RS puis appuyez sur le bouton Executer.

Racket

  • Commandes d'exécution : executer, stopper
  • Indenter du texte
  • Commenter ou décommenter un texte selectionné

3. La fenêtre de définitions

L'éditeur de DrRacket

  • gère les parenthèses ;
  • fait l'indentation, automatiquement, à chaque changement de ligne (avec la touche  Entrée).

La barre d'outils

  • le bouton Sauvegarder(qui n'apparaît que lorsque le contenu de la fenêtre de définitions a été changé) permet de sauvegarder ces définitions dans le fichier dont le nom apparaît à gauche du bouton; une fenêtre de dialogue apparaît lors de la première sauvegarde (pour que le nom du fichier soit donné). Après que le fichier est sauvegardé, ce bouton disparaît;
  • le bouton Vérifier permet de vérifier la syntaxe du programme contenu dans la fenêtre;
  • le bouton Déboguer permet de contrôler l'exécution du programme. Il affiche également le contenu de la pile d'exécution et les variables définies. Il est utile pour chercher les erreurs. Il permet de mettre l'exécution en pause (Pause),  de relancer l'exécution (Go) de visualiser, pas à pas, les étapes d'évaluation des expressions (Step), de gérer des points d'arrêt (Over, Out) ;
  • le bouton Executer permet d'évaluer les expressions contenues dans la fenêtre de définitions, ou de réinitialiser la fenêtre d'interactions ;
  • le bouton Stopper permet d'interrompre l'exécution d'un programme.

4. La fenêtre d'interactions

La fenêtre d'interactions permet d'écrire une expression après le prompt (>) ; cette expression sera immédiatement évaluée; le résultat est écrit dans une nouvelle ligne; ensuite un nouveau prompt apparaît, pour permettre d'écrire une nouvelle expression (remarque: l'expression peut être écrite sur plusieurs lignes)

On ne peut pas modifier une expression déjà évaluée.

5. Utilisation de DrRacket

Évaluation d'expressions

Dans la fenêtre d'interactions, évaluez l'une après l'autre chacune des 9 expressions suivantes 

  1. (+ 4 5) -->
  2. (sqrt 25) -->
  3. 365 -->
  4. 'test -->
  5. test -->
  6. (define test 4.25) -->
  7. test -->
  8. (and (> 12 45) (= 3 (/ 12 4) ) ) -->
  9. (expt 2 3) -->

Définition de fonctions

Dans la fenêtre d'interactions, tapez la définition de fonction suivante :

(define (double x) (* 2 x))

Evaluer quelques exemples d'application de la fonction double

Charger un fichier de définitions (programme)

  1. Enregistrer le fichier exemple1.ss dans un répertoire
  2. Ouvrir le fichier sous DrRacket et l'évaluer
  3. utiliser le bouton Step du débogueur pour évaluer pas à pas le programme
  4. Evaluer quelques exemples d'application de la fonction hypoténuse. Notez qu'une fonction peut être définie soit en écrivant sa définition dans la fenêtre d'interaction soit en créant un fichier qui contient sa définition et en évaluant ce fichier
  5. Enregistrer le fichier exemple2.ss dans un répertoire
  6. Ouvrir et exécuter ce fichier.

Programmation de fonctions

  1. Définir une fonction f permettant de calculer :

            f(x,y) =   x - 3    +      1 
                       y2 + 1       x3 + 3

  2. Définir une fonction cel->far effectuant la conversion en degrés Fahrenheit d'une température donnée en degrés Celsius selon la formule F= 9/5 C + 32
  3. Définir une fonction distance qui calcule la distance entre deux points A et B de coordonnées respectives (xA, yA), (xB, yB)
  4. Conversion horaire
    1. Définir une fonction m->s permettant de convertir des minutes en secondes
    2. Définir une fonction h->m permettant de convertir des heures en minutes
    3. Utiliser les deux fonctions précédentes pour définir une fonction h->s permettant de convertir en secondes un horaire exprimé en heures, minutes et secondes
  5. Définir une fonction moy-pond calculant la moyenne pondérée de 2 notes n1 et n2 affectées respectivement des coefficients c1 et c2.
