Exercice 3

Définir la fonction impots calculant l'impôt brut sur le revenu, connaissant le revenu net imposable R et le nombre de parts N, en utilisant la méthode suivante (barème 2017 de la déclaration des revenus 2016) :

Soit QF = R / N le quotient familial. Il y a cinq fractions d'imposition pour une part de quotient familial. 


QF <= 9710 €                 : le taux d'imposition est de 0 %
9710 € < QF <= 26818 €       : le taux d'imposition est de 14 %
26818 € < QF <= 71898 €      : le taux d'imposition est de 30 %
71898 € < QF <= 152260 €     : le taux d'imposition est de 41 %
152260 € < QF                : le taux d'imposition est de 45 %

Pour calculer l'impôt, il faut calculer les montants par fraction, les additionner,  puis multiplier le résultat par N.

Exemple : R = 54000, N = 2, QF = 27000

Fraction jusqu'à 9710 €         : 0 €
Fraction de 9710 € à 26818 €    : (26818 - 9710) * 0,14 = 2395,12 €
Fraction de 26818 à 27000 €     : (27000 - 26818) * 0,30 = 54,6 €

Montant de l'impôt brut : (0 + 2395,12 + 54,6) * 2 = 4899,44 €

Exercice 4

Pour calculer la date de Pâques d’une année donnée, on calcule le nombre de jours entre le 1er mars et Pâques, et on en déduit le mois (mars ou avril) et le jour de Pâques.

1) Définir la fonction nbj-mars calculant le nombre de jours à compter du 1er mars en fonction d’une année donnée.


> (nbj-mars 1998)
43

Le nombre de jours à compter du 1er mars est obtenu par la suite de calculs :

  • a = Année modulo 19
  • b = Année modulo 4
  • c = Année modulo 7
  • d = ( 19 a + 24 ) modulo 30
  • e = ( 2 b + 4 c + 6 d + 5 ) modulo 7

Le nombre de jours à compter du 1er mars s’obtient alors par la formule : 22 + d + e

2) Définir la fonction mois donnant le mois de Pâques (mars ou avril) en fonction du nombre de jours à compter du 1er mars.


> (mois 43)
"avril"

3) Définir la fonction jour donnant le jour (du mois) de Pâques en fonction du nombre de jours à compter du 1er mars.


> (jour 43)
12

4) Définir la fonction affiche-paques qui utilise les fonctions précédentes pour afficher la date complète du jour de Pâques en fonction d’une année donnée.


> (affiche-paques 1998)
12 avril 1998
